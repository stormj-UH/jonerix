# Package build order — dependency + size ordered
#
# Dependency rules:
#   musl      → (no deps, must be first)
#   ncurses   → (no deps)
#   libressl   → musl
#   zstd      → musl
#   libarchive→ musl, libressl, zstd, lz4, xz
#   bsdtar    → libarchive
#   curl      → musl, libressl
#   python3   → musl, libressl
#   unbound   → musl, libressl
#   nodejs    → musl, zlib
#   cmake     → musl, clang, samurai, python3
#   go-bootstrap → musl, clang (x86_64 only)
#   go        → musl, clang, go-bootstrap
#   go-current→ musl, clang, go
#   rust      → musl, llvm, rust (prebuilt), cmake, samurai, python3, curl
#   ripgrep   → musl, rust
#   uutils    → musl, rust
#   anvil     → musl, rust
#
# Within each dependency tier, packages are ordered smallest-first
# so that interrupted builds leave the most useful artifacts.
#
