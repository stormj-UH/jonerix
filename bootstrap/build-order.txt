# Bootstrap build order — dependency + size ordered
#
# Dependency rules:
#   musl    → (no deps, must be first)
#   ncurses → (no deps)
#   openssl → musl
#   zstd    → musl
#   pico    → ncurses
#   curl    → musl, openssl
#   python3 → musl, openssl
#   unbound → musl, openssl
#   nodejs  → musl, zlib (zlib from pre-bootstrap)
#   pigz    → musl, zlib (zlib from pre-bootstrap)
#
# Within each dependency tier, packages are ordered smallest-first
# so that interrupted builds leave the most useful artifacts.
#
